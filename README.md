## Entradas Online - Sistema de Venta de Entradas

Sistema web completo para la venta de entradas de conciertos y eventos en vivo, desarrollado con HTML, CSS, JavaScript, PHP y MySQL.

## Características

- Diseño moderno y responsive (mobile-first)
- Panel de administración completo
- API RESTful para gestión de datos
- Sistema de registro y autenticación de usuarios
- Gestión de conciertos, entradas y usuarios
- Validaciones en frontend y backend
- Seguridad: contraseñas encriptadas, prepared statements
- Animaciones y micro-interacciones

## Tecnologías

### Frontend
- HTML5
- CSS3 (con animaciones personalizadas)
- JavaScript (ES6+)
- Bootstrap 5.3.6
- Google Fonts (Poppins)
- Font Awesome 6.5.0

### Backend
- PHP 8.1+
- MySQL
- API RESTful
- JSON

### Herramientas
- Composer (gestión de dependencias)
- Symfony Framework (opcional)
